# Abhishek Travels - South India Travel Website

A modern, responsive travel website for exploring destinations and booking accommodations across South India.

## Features

- **Destinations**: Explore 20+ destinations across 5 South Indian states
  - Karnataka (Bangalore, Mysore, Coorg, Hampi, Gokarna, Chikmagalur)
  - Tamil Nadu (Chennai, Ooty, Kodaikanal, Madurai, Rameswaram, Kanyakumari)
  - Kerala (Munnar, Alleppey, Kochi, Wayanad, Thekkady, Kovalam)
  - Andhra Pradesh (Visakhapatnam, Tirupati, Araku Valley, Vijayawada)
  - Telangana (Hyderabad, Warangal)

- **Room Booking**: Search and book rooms across various destinations
  - Filter by destination
  - Select check-in/check-out dates
  - Choose number of guests and rooms
  - View room details and amenities
  - Complete booking process

- **Modern UI/UX**:
  - Responsive design for all devices
  - Smooth scrolling navigation
  - Interactive destination filtering
  - Beautiful hero section
  - Contact form and newsletter subscription

## How to Use

1. **Open the Website**: Simply open `index.html` in your web browser
2. **Explore Destinations**: Click on "Destinations" to see all South India locations
3. **Filter by State**: Use the filter buttons to view destinations by state
4. **Book Rooms**: 
   - Go to "Rooms" section
   - Select destination, dates, guests, and rooms
   - Click "Search Rooms"
   - Click "Book Now" on any room
   - Fill in booking details and confirm

## Customization

You can easily customize:
- **Colors**: Edit CSS variables in `styles.css` (`:root` section)
- **Destinations**: Add/modify destinations in `script.js` (`destinations` array)
- **Rooms**: Add/modify rooms in `script.js` (`rooms` array)
- **Contact Info**: Update contact details in `index.html` (Contact section)

## Notes

- This is a frontend-only website. For production use, you'll need to integrate with a backend for actual booking processing.
- Images are loaded from Unsplash. For production, consider hosting your own images.
- Booking confirmations are currently simulated with alerts. Integrate with a payment gateway and booking system for real functionality.
